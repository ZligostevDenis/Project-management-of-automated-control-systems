2. Разработка программного обеспечения для автоматизации аттестации персонала IT компании.
3.Разработка программного обеспечения системы учёта развития спортсменов для МБУДО
4.Геоинформационная система литературного картирования произведений русской литературы по роману Л.Н. Толстого «Война и мир».

8.Разработка программного обеспечения задачи автоматизации обработки заявлений на получение социальных пособий гражданами Пермского края.
9. Разработка программного обеспечения задачи автоматизации системы проверок объектов топливно-энергетического комплекса для внутреннего приложения компании ООО «ЛУКОЙЛ-Технологии».
